# Avdesh Jadon - Portfolio

🌐 **Live Site:** [https://avdeshjadon.dpdns.org/](https://avdeshjadon.dpdns.org/)

A modern, minimal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. 

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/avdeshjadon/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
