# Avdesh Jadon - Portfolio

Live Site: [https://avdeshjadon.dpdns.org/](https://avdeshjadon.dpdns.org/)

A modern, minimal, and highly interactive portfolio website built to showcase projects, skills, and professional experience. The application features 3D elements, smooth animations, and a fully responsive design, ensuring an optimal viewing experience across all devices.

## Features

- Immersive 3D Graphics: Integrates React Three Fiber and Drei for interactive 3D illustrations.
- Smooth Animations: Utilizes Framer Motion for seamless page transitions and scroll-linked animations.
- Dynamic Typing Effects: Features customized typing animations for engaging text presentations.
- Responsive Layout: Built with Tailwind CSS to ensure a mobile-first, adaptive user interface.
- Optimized Performance: Powered by Vite for rapid development and optimized production builds.

## Technologies Used

- Framework: React (v19)
- Build Tool: Vite
- Styling: Tailwind CSS
- Animation: Framer Motion
- 3D Rendering: Three.js, React Three Fiber, React Three Drei
- Icons: React Icons

## Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

You need Node.js and npm installed on your system before proceeding.

### Installation

Clone the repository to your local machine:

```bash
git clone https://github.com/avdeshjadon/portfolio.git
```

Navigate into the project directory:

```bash
cd portfolio
```

Install the project dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at http://localhost:5173.

## Available Scripts

In the project directory, you can run the following commands:

- `npm run dev`: Runs the application in development mode.
- `npm run build`: Builds the application for production to the output folder.
- `npm run lint`: Evaluates the codebase using ESLint to ensure code quality.
- `npm run preview`: Locally previews the production build.
